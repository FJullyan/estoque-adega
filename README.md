# 🍷 Adega Stock - Sistema de Controle de Estoque

Sistema completo de controle de estoque para adegas e comércios em geral.

## Links
- **Frontend:** https://estoque-adega.vercel.app
- **Backend:** https://estoque-adega.onrender.com

## Funcionalidades
- Cadastrar, editar e deletar produtos
- Buscar por nome ou categoria
- Registrar entradas e saídas de estoque

## Tecnologias
- Python + Flask
- Supabase (banco de dados)
- HTML, CSS e JavaScript
- Bootstrap
